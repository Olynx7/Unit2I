# Unit2I Plan 文档说明

这个目录主要用于保存项目最开始阶段的规划文档（plan），不是最终用户文档。

## 当前文档分工
- [DESIGN.md](DESIGN.md)：设计方法、API 规范、参数与错误语义。
- [EXAMPLES.md](EXAMPLES.md)：包实现后可以调用的方法和示例代码。

## 当前厂商范围（v0.1）
- 阿里 DashScope
- 字节火山引擎 Volcengine

## 版本路线（建议）
- v0.1：先交付可用最小版本（双 provider + 同步生成 + 基础批量）。
- v0.2：再扩展异步任务、本地文件输出、更多 provider。

## 备注
- 这里是规划层文档，后续如果厂商或 API 有调整，以 DESIGN 为准并同步更新 EXAMPLES。
